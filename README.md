# COVID Kokua by Team WAP

## Run Locally
1. [Setup your environment for Node.js developement](https://nodejs.org/).

2.  Clone this repo:

        git clone https://github.com/HACC2020/WAP.git

3.  Install depedencies:

        npm install

4.  Run the with `npm`:

        npm start

5.  Visit the application at [http://localhost:8080]().
